# Contributing to Remotastic

## Setup

1. Install Xcode Command Line Tools: `xcode-select --install`
2. Clone and build:
   ```bash
   git clone https://github.com/yourusername/remotastic.git
   cd remotastic
   ./build.sh
   ./create_app_bundle.sh
   ```
3. Grant Accessibility permissions in System Settings

## Code Style

- Follow Swift conventions
- Use meaningful names
- Keep functions focused
- Prefer `let` over `var`

## Submitting Changes

1. Fork and create a feature branch
2. Make changes following code style
3. Test with a real Siri Remote
4. Submit a PR with a clear description

## Areas for Contribution

Bug fixes, performance improvements, new features, documentation, and testing.
